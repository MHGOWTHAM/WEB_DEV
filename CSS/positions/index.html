Positions in css

  • Static : The original or default poition of an element.
  • Relative : Element is positioned relative to it's normal position.same as static positioning, we can able to move the element by setting the offset values. Element stays in normal doc flow.
  • Absolute : Element is positioned relative to it's parent. If the parent is not present the element is position relative to its main document.Element is removed from the normal document flow, no space is created for that element.
  • Fixed : Element is positioned relative to it viewport. Even we scroll down page,the element stays in it's same position.
  • Sticky : Element is positioned relative to it's normal document flow.Till the element reach the top remains same, after it touches the top it becomes fixed.
