Transition : Allows u to change the property values smootly

  1. transition : Shorthand property
  2. transition-delay: It specifies when the transition effect will start.
  3. transition-duration: It specifies the how much time it should lost
  4. transition-property : It specifies the css property to which you want to apply transition.
  5. transition-timing-function : It specifies the speed curve of the transition effect
    1. ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier(n,n,n,n)
